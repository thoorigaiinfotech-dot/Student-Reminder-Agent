# Student Reminder Agent

An AI-powered student reminder system built using n8n.

## Features

- Automatic follow-up emails
- AI-generated email content using OpenAI
- Google Sheets integration
- Gmail integration
- Batch reminder automation
- Scheduled daily execution

## Technologies Used

- n8n
- OpenAI GPT
- Google Sheets API
- Gmail API

## Workflow

Schedule Trigger
↓
Google Sheets
↓
JavaScript Logic
↓
IF Node
↓
OpenAI
↓
Gmail
↓
Update Google Sheet
